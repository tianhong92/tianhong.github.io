monit报警邮件配置

monit配置报警邮件， 现在以qq smtp服务为例。

根据官方文档配置mailserver

set mailserver smtp.qq.com port xx
username "qq号" password "不是密码是授权码"
using SSL

配置mail-format

set mail-format {
    from: 454648xx@qq.com
    subject: monit alert -- $EVENT
    message: Date: $DATE
         Description: $DESCRIPTION
 }
